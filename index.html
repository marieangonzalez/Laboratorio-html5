Sintaxis básica de HTML

HTML significa HyperText Markup Language. Se utiliza para crear y organizar el contenido de una página web.

Los elementos normalmente contienen una etiqueta de apertura, contenido y una etiqueta de cierre.

Ejemplos de etiquetas

  • <h1> crea un encabezado principal.
  • <p> crea un párrafo.
  • <ul> crea una lista sin orden.
  • <ol> crea una lista ordenada.
  • <a> crea un enlace.

Semántica de HTML5

¿Qué es la semántica?

La semántica se refiere al significado de cada elemento. Las etiquetas semánticas indican qué función cumple una sección dentro de la página.

Elementos semánticos utilizados

  1. Header: representa el encabezado.
  2. Nav: contiene los enlaces de navegación.
  3. Main: contiene el contenido principal.
  4. Section: divide el contenido por temas.
  5. Article: contiene información independiente.
  6. Footer: representa el pie de página.

Características multimedia de HTML5

Elemento de video

Ventajas de la estructura HTML5

  • Organiza mejor el contenido de la página.
  • Permite identificar la función de cada sección.
  • Mejora la accesibilidad para los usuarios.
  • Facilita el mantenimiento del código.
  • Permite incorporar audio, video e imágenes.
  • Ayuda a los navegadores y motores de búsqueda a interpretar la página.
