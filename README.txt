Descripción:

En este laboratorio se creó una página web utilizando Visual Studio
Code. La página presenta la sintaxis básica de HTML, elementos y
atributos, encabezados, párrafos, listas, enlaces y una estructura
semántica de HTML5.

Se utilizaron las etiquetas header, nav, main, section, article,
figure, figcaption y footer para organizar el contenido. También se
incorporaron los elementos audio, video e img.

El archivo estilos.css se utilizó para modificar los colores, espacios,
tipografía, navegación y presentación visual de la página.

La página fue visualizada y probada en Google Chrome. Finalmente, los
archivos fueron guardados en un repositorio de GitHub para conservar
una copia en línea del proyecto.

Archivos principales:

- index.html
- estilos.css
- README.txt
- Carpeta imagenes
- Carpeta audio
- Carpeta video
- Video de evidencia en formato MP4
